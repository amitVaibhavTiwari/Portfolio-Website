# Developer Portfolio Template

A clean, minimal portfolio template built for developers. This is the template I use for my own personal website [amitvaibhavtiwari.dev](https://www.amitvaibhavtiwari.dev).

### Prerequisites

- Node.js 18+
- npm / yarn / pnpm

### Installation

```bash
git clone https://github.com/amitVaibhavTiwari/Portfolio-Website.git
cd Portfolio-Website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Deployment Options

### Option 1 — Next.js Supported Platforms

Push your code to GitHub and deploy directly to any Next.js supported platform such as Vercel or Netlify.

### Option 2 — Static Export (Any Host)

Run `npm run build` to generate static assets (HTML, CSS and JS) in the `out/` directory. Upload that folder to any static hosting provider — GitHub Pages, Cloudflare Pages, an S3 bucket, or any web server.

---

## Customization

1. Update personal details, links, and content in `app/page.tsx`
2. Replace `public/home_img.jpg` with your own profile photo
3. Edit `app/layout.tsx` for metadata (title, description, Open Graph)
